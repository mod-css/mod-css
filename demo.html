Demo MOD-CSS

MOD-CSS is a free utility-first CSS framework for front-end development.
It's build only with javascript and supports breakpoints, flexgrid, states and CSS properties.

Design from scratch directly in your template, anything you want.
We support almost all CSS properties.

Only single file (10kb) to get all power of CSS properties.

Download here

Basic Usage

Set arbitrary value on any css properties supported by MOD CSS.
Call CSS Propeties by its codename in the right attribute.

Flexbox grid

Our flexbox based grid can use any css units, can setting gap and offset, automatic sizing, custom width etc...
It is probably the most complete flexbox based grid ever created.

Breakpoints

We have six breapoints : xs, sm, md, lg, xl, xxl. They can be used for responsiveness with all mod css properties as well as with pseudo-classes and our grid.

Pseudo-States

You can also use focus, active and all others pseudos-classes.
